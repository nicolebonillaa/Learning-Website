Our lessons!

Explore our wide range of lessons designed to help you learn and grow. Whether you're interested in programming, data structures, algorithms, or formal logic, we have something for everyone. Start your learning journey today!

Data Structures

5 quizzes covering Trees, Binary Trees, AVL Trees, Hash Tables, and Abstract Data Types. Learn how data is organized, stored, and accessed efficiently.

Java Programming

5 quizzes on Variables, Operators, Strings, Booleans, and Loops. Everything you need to start writing Java confidently.

Sorting Algorithms

6 quizzes on Merge Sort, Radix Sort, Selection Sort, Bubble Sort, Heap Sort, and Insertion Sort. Understand how and when to use each one.

Formal Logic

7 quizzes spanning Propositions, Negation, Conjunction, Disjunction, Truth Tables, XOR, and Biconditionals. The foundation of logical reasoning.
