Formal Logic

Formal logic is a branch of mathematics and philosophy that studies the principles of valid reasoning and inference. It provides a systematic framework for analyzing arguments and determining their validity. Formal logic uses symbolic notation to represent logical statements and relationships, allowing for precise analysis and manipulation of logical expressions.

Propositional Logic

Propositional logic, also known as sentential logic, is a branch of formal logic that deals with propositions and their relationships. It focuses on the logical connectives (such as AND, OR, NOT) and the rules of inference that govern how propositions can be combined and manipulated to form valid arguments.

Negation

Negation is a fundamental operation in formal logic that takes a proposition and produces its opposite. It is denoted by the symbol "¬" or "!" and is used to express the denial or contradiction of a statement. For example, if the proposition "P" is true, then its negation "¬P" would be false.

Conjunction

Conjunction is a logical operation that combines two propositions and produces a new proposition that is true only if both of the original propositions are true. It is denoted by the symbol "∧" or "&&". For example, if we have two propositions "P" and "Q", the conjunction "P ∧ Q" would be true only if both "P" and "Q" are true.

Disjunction

Disjunction is a logical operation that combines two propositions and produces a new proposition that is true if at least one of the original propositions is true. It is denoted by the symbol "∨" or "||". For example, if we have two propositions "P" and "Q", the disjunction "P ∨ Q" would be true if either "P" or "Q" (or both) are true.

Truth Tables

Truth tables are a tool used in formal logic to analyze the truth values of logical expressions. They list all possible combinations of truth values for the propositions involved and show the resulting truth value of the expression for each combination. Truth tables are particularly useful for understanding the behavior of logical connectives and determining the validity of arguments.

XOR

XOR, or exclusive OR, is a logical operation that combines two propositions and produces a new proposition that is true if exactly one of the original propositions is true. It is denoted by the symbol "⊕" or "^". For example, if we have two propositions "P" and "Q", the XOR "P ⊕ Q" would be true if either "P" or "Q" is true, but not both.

Biconditionals

Biconditionals, also known as logical equivalence, is a logical operation that combines two propositions and produces a new proposition that is true if both of the original propositions have the same truth value (both true or both false). It is denoted by the symbol "↔" or "<->". For example, if we have two propositions "P" and "Q", the biconditional "P ↔ Q" would be true if both "P" and "Q" are true or if both "P" and "Q" are false.
